# 🏁 地平线 6 · 自动助手 v1.1

一个 **Forza Horizon 6** 的挂机自动助手:自动买车加点 + EventLab 刷图循环、拍卖场抢车、清理重复车、一键买光未拥有的车。**只通过虚拟 Xbox 手柄操作 + 只读截屏识别,不注入游戏、不修改任何游戏文件。**

> 直接下载下面的 `Forza6HelperV4GUI.exe`,双击运行即可,无需安装 Python。

## 🔄 v1.1 更新内容(相比 v1.0)
- **适配简体中文更新**:游戏把「车辆熟练度」改名为「车辆专精」,**买车加点流程已兼容**(已在游戏内实测通过)。
- **「买未拥有的车」长时间挂机更稳,不再卡死**:
  - 手柄掉线(弹「控制器未连接」)会**自动按 A 重连**并继续。
  - 卡住自愈:万一漂移到异常画面,会**自动退回、重新导航回『车展』网格页**继续买。
  - 加了超时看门狗,并把运行日志存到程序旁边的 `logs/gui.log`(方便出问题时排查)。

## ✨ 功能一览

| 功能 | 说明 |
|---|---|
| **模式三循环** | 买车(斯巴鲁 22B)+ 加满车辆熟练度 → 导航到 EventLab 刷分赛 → 视觉刷图 → 收尾,如此无限循环。可选「先买车再跑图」或「先跑图再买车」。技术点数耗尽会自动关掉提示继续,不再卡死。 |
| **拍卖场抢车** | 在「搜寻」页设好车型 + 最高买断价,程序不停重搜,目标一出现就买断(**只买断、绝不竞价**)。 |
| **清理重复车** | 用游戏自带「重复项」筛选,批量删掉多余的某型号车(自动**保留收藏的 / 正在驾驶的**那辆,删前核对确认框)。 |
| **买未拥有的车** | 在车展自动勾上「未拥有」筛选,一辆辆把没有的车全买下来,直到买完。 |
| **把地平线调成 16:9** | 一键把游戏窗口规范成 16:9,让识别在任意显示器(含带鱼屏)上都稳定。 |

> 每个功能都有独立的「停止」键,随时可停;数量不设上限,跑到完成或你手动停为止。

## 🖥️ 全新界面(本版本重点)
- **横向深色现代界面**,运行日志/报告在**右侧整条**显示。
- 大字体、扁平按钮、深色标题栏与窗口融为一体,整体更清爽。

## 📦 怎么用
1. 下载 **`Forza6HelperV4GUI.exe`**,双击运行(首次打开约十几秒,在加载识别模型)。
2. 需要 **ViGEmBus** 虚拟手柄驱动;没装的话程序会提示并自动打开下载页,装完重开即可。
3. **游戏建议用「窗口 / 无边框窗口」模式**,运行时保持 Forza 在前台(失焦会自动切回)。
4. 各功能的起始页面:
   - **模式三**:停在**暂停菜单**,点「开始」。
   - **拍卖抢车**:进 拍卖场 → 搜索拍卖,设好过滤,停在**「搜寻」配置页**。
   - **清理重复**:停在**我的车辆 / 车辆标签页**。
   - **买未拥有**:进 车辆 → 购买车与二手车 → 车展,停在**车辆网格页**。

## 🛡️ 原理与安全
- **只发标准虚拟 Xbox 手柄输入**(通过 ViGEmBus),和你自己拿手柄按没有区别。
- **只读截屏 + 本地识别**(YOLO / OCR / 规则),**不读写游戏内存、不注入、不修改任何游戏文件**。
- 需要游戏保持前台(失焦时自动切回,属于正常的窗口切换)。
